# Smart Loan Approval System

This project is an AI-based system designed to automate loan approval decisions using machine learning.

## Description
The system analyzes financial data such as income, loan amount, and credit score to determine whether a loan should be approved or rejected.

## Models Used
- Random Forest Classifier
- Deep Learning (Neural Network)

## Technologies Used
- Python
- Machine Learning
- Data Analysis

## Features
- Automated loan decision making
- Risk assessment based on financial data
- Comparison between different AI models
